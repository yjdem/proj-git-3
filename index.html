      Краткое содержание:
      демонстрация работы основных команд в командной строке Git;
      инициализация проекта, добавление/удаление содержимого в индекс коммита, создание коммита, проверка состояния.

      Цель занятия:
      повторить как использовать базовые команды для работы с системой Git.

      Для начала необходимо на ПК установить GIT - https://selectel.ru/blog/tutorials/how-to-install-git-to-windows


      С помощью командной строки выполнить следующее:

      1. Настройка Git
      2. Создание проекта
      3. Добавление файлов в индекс

      5. Удаление файлов из индекса
      6. Создание коммита
